First, let's define the functions for `sin(x)` and `saw(x)`. The sine function can be directly used from the JavaScript Math library, and for the sawtooth wave, we will implement it manually using a formula that provides the sawtooth wave characteristic.

Next, for the predicates `pressL(e)` and `pressR(e)`, we will use JavaScript's event listener mechanism to listen for keyboard events and react accordingly.

Here is the JavaScript code for these implementations:

### Functions

```javascript
// Function to calculate the sine of x
function sin(x) {
    return Math.sin(x);
}

// Function to calculate the sawtooth wave of x
function saw(x) {
    // Sawtooth wave formula: x - floor(x)
    return x - Math.floor(x);
}
```
